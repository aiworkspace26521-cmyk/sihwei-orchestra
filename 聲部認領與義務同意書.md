# 高雄市四維國小管弦樂團家長後援會
### ── 團員入團聲部認領與後勤義務同意書 ──

親愛的家長您好：
歡迎您的孩子加入「高雄市四維國小管弦樂團」！樂團不僅是孩子學習音樂的殿堂，更是一個需要全體家長共同投入、協力維繫的自治社團。

為了確保年度兩大專案—**「11月全國音樂比賽」**與**「6月年度成果發表會」**能圓滿成功，且平日團練安全順暢，本後援會採行**「日常聲部自治」與「大活動聲部承包」**機制。每位家長依孩子就讀聲部，自動成為該聲部組員，共同分擔後勤工作，並在活動期間服從後援會**「活動統籌組」**的調度與指揮。

請您詳閱以下聲部職責，並填寫下方的認領同意書交回樂團，感謝您的熱心參與！

---

### 📌 各聲部大活動承包職責對照表

| 團員就讀聲部 | 大活動轉職承包組別 | 當日核心職責與戰位 |
| :--- | :--- | :--- |
| 🎻 **弦樂一部** <br>(小提琴一) | **【後台照護與生活後勤組】** | 守在後台休息室。負責學生餐點發放、換裝/化妝協助、維護休息區秩序與安全。 |
| 🎻 **弦樂二部** <br>(小提琴二、中提琴) | **【前台公關與票務宣傳組】** | 守在演出場館大廳。負責前台索票與驗票、貴賓接待與引導。公演前負責海報/節目單美術編排。 |
| 🎸 **大提琴 / 低音提琴 / 打擊樂部** | **【舞台硬體與物流組】** | 守在卸貨區與舞台。負責大件樂器裝箱搬運、貨車裝卸，及舞台椅位、譜架之催台與定位。 |
| 🎺 **管樂部** <br>(長笛、單雙簧管、銅管等) | **【機動調度與廠商對接組】** | 守在側台與機動區。負責催台引導、機動協助廠商拉電線與擺放道具、協助現場流程掌控。 |

*   **全體家長平日義務：** 所有聲部家長皆須配合**【學務組】**的安排，參與每學期 1-2 次（每次 2 小時）的平日團練排練室家長義務值班。
