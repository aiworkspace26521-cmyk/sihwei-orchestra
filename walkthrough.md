## 🏛 組織架構與運作機制摘要

本方案最大的特色是**「日常依聲部社交自治，活動時按場地整組變形承包」**：

```
                      ┌──────────────────────────────────────┐
                      │            後援會會長 (1人)          │
                      └──────────────────┬───────────────────┘
                                         │
               ┌─────────────────────────┼─────────────────────────┐
               ▼                         ▼                         ▼
   ┌──────────────────────┐  ┌──────────────────────┐  ┌──────────────────────┐
   │    財務組 (2人)      │  │  活動統籌組 (1-2人)  │  │    學務組 (1-2人)    │ <--- 後援會三大支柱 (日常行政)
   │  - 總帳記帳與透明公開│  │  - 跨部門調度與PMO   │  │  - Master Database   │
   │  - 各部團費與鐘點核銷│  │  - 外部廠商合同對接  │  │  - 學校行政與公假保險│
   └──────────────────────┘  └───────────┬──────────┘  └───────────┬──────────┘
                                         │                         │
                                    (活動現場指揮)             (平日行政匯報)
                                         ▼                         ▼
  ┌───────────────────┬──────────────────┴──────────────────┬───────────────────┐
  ▼                   ▼                                     ▼                   ▼
┌──────────────┐    ┌──────────────┐                      ┌──────────────┐    ┌──────────────┐
│  弦樂一部組  │    │  弦樂二部組  │                      │ 大/低音提琴部│    │  管樂打擊部  │ <--- 四大聲部日常自治
│ (小提琴一)   │    │(小&中提琴)   │                      │  (大/低提)   │    │  (管樂打擊)  │
├──────────────┤    ├──────────────┤                      ├──────────────┤    ├──────────────┤
│ 📍活動轉職： │    │ 📍活動轉職： │                      │ 📍活動轉職： │    │ 📍活動轉職： │
│【後台照護組】│    │【前台公關組】│                      │【舞台物流組】│    │【機動對接組】│ <--- 大活動聲部承包
└──────────────┘    └──────────────┘                      └──────────────┘    └──────────────┘
```

*   **平日（聲部自治）：** 家長在所屬的樂器聲部內點名、請假、輪流排班值班。
*   **大活動當日（聲部承包）：** 
    *   **大提琴/低提/打擊家長** 整組進入 **舞台與側台**，搬運重型樂器與擺放椅位譜架（因為他們最懂這些樂器）。
    *   **小提琴一部家長** 整組進入 **後台休息室**，管理後台秩序、發放便當、協助化妝換裝（人數最多，最能負擔繁重照護）。
    *   **小提琴二部/中提琴家長** 整組進入 **劇院大廳前台**，索票驗票、貴賓接待。
    *   **管樂家長** 負責 **側台機動與技術廠商對接**。
*   **活動統籌組：** 作為現場大腦，統一發號司令，調度以上各聲部代表，會長則專注於學校行政與大局決定。

## 🚀 後續實推行指南 (Roll-out Strategy)

為了讓這套系統順利落地，建議依循以下步驟實施：

1.  **校方與指揮共識：** 將 [四維國小管弦樂團營運說明書.md](file:///d:/20260605 Antigravity/四維國小管弦樂團/四維國小管弦樂團營運說明書.md) 提交給學校輔導室與指揮周老師，確認學校行政配合度與排練值班安排。
2.  **大會發放與認領：** 於學期初家長大會中發放 [聲部認領與義務同意書.md](file:///d:/20260605 Antigravity/四維國小管弦樂團/聲部認領與義務同意書.md)。說明「聲部自治」與「大活動聲部承包」的用意，讓每位家長勾選簽字交回。
3.  **登錄雲端地圖：** 依據收回的同意書，由學務組長建立全團通訊錄，並將各家長編入後援會人力地圖中，空缺位置由會長協調補齊。
4.  **建立雲端記帳與值班試算表：** 依照說明書第六章，建立唯讀的 Google 記帳試算表與值班登記表，發布於 Line 群組公告。
