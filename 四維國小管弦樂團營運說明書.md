## 一、 樂團管理願景與核心原則

高雄市四維國小管弦樂團成立於1999年，是由校內喜愛音樂的孩子們所組成。為確保樂團在有限的家長志工人力下，能高效推進樂團事務，並在「年度成果發表會 (6月)」與「全國學生音樂比賽 (11月)」取得卓越成果，本手冊依循以下四大管理原則編撰：

1.  **以始為終 (Begin with the End in Mind)：** 
    所有日常行政、排練安排、資源配置，皆以年度兩大成果發表（11月 比賽、6月 至德堂公演）為專案終點，進行倒推排程，確保所有準備工作在關鍵節點（Milestones）準時完成。
2.  **扁平化與去中心化：**
    取消副會長等虛設職務，由「會長」直轄三大行政組，進行極度扁平化的溝通與敏捷決策。
3.  **三大行政支柱：**
    建立「財務組 (掌管錢)」、「活動統籌組 (掌管事/物/廠商/現場調度)」、「學務組 (掌管人/學校行政/招生)」三大行政中樞，提供樂團運作的穩固日常架構。
4.  **全員認領與聲部承包：**
    家長依據孩子所就讀之樂器聲部，自動歸入所屬聲部，平日進行「聲部自治」；大型活動當天，各聲部整組變形轉職，集體承包特定的後勤區域，消除「我只是來幫忙的」旁觀者心態，實現 100% 的主體責任感。

> 💡 **樂團年度運作「雙核心里程碑」導讀**
> 後援會以始為終的所有專案排程與家長承包制度，皆環繞著年度兩大高峰任務運作：
> * **【11月 全國學生音樂比賽（市賽）】** ➔ 樂團首度出征，家長啟動大活動後勤承包，全力護航。
> * **【6月 年度成果發表會（公演）】** ➔ 樂團年度最大盛事，至德堂公演現場全員變形承包。
>

## 四、 三大行政支柱（中間層）詳細職責

### 1. 財務組 (Finance Team)
*   **定位：** 樂團資金的守护者，確保每一分錢收支均有跡可循。
*   **核心工作：**
    *   **預算控管：** 於每學期初，協同會長與指揮老師，編列學學期總預算書。
    *   **團費收取：** 開學時發放繳費通知，負責收取團費、寒暑假集訓費，並確實開立收據。
    *   **出納與主計分離：** 出納崗負責保管現金與實體銀行帳戶；主計崗負責登錄雲端帳冊、審核收據與發票。
    *   **財務透明化：** 每月最後一天，主計崗需將「雲端收支明細表（PDF檔）」發布至家長大群組，供全體家長檢視。
    *   **師資付款：** 每月依據學務組提供之出勤紀錄，結算並匯款給指揮及外聘分部指導老師。

### 2. 活動統籌組 (Events Coordination Team)
*   **定位：** 樂團的專案管理辦公室 (PMO)，對外負責廠商談判，對內負責跨部門調度。
*   **核心工作：**
    *   **專案時程管理：** 將比賽與音樂會拆解為專案進度表，每月追蹤各聲部代表的進度。
    *   **外部廠商對接：** 負責與高雄文化中心至德堂等場地對接；負責大貨車、遊覽車、便當、音響、攝錄影廠商的訪價、比價與合約簽署。
    *   **大活動現場指揮官：** 在公演與比賽當日，與場方的「舞台監督」對接，控制彩排及演出時間，並在側台指揮、調度四大聲部所承包的後勤功能。

### 3. 學務組 (Academic & Student Affairs Team)
*   **定位：** 樂團的「學籍與行政總務」，負責所有涉及「人與文件」的工作。
*   **核心工作：**
    *   **Master Database維護：** 建立全團學生基本資料庫（含身份證字號、出生年月日，用於比賽報名與保險）。
    *   **出缺席管理：** 彙整各聲部每日練習的點名表，統計團員請假紀錄。
    *   **學校行政窗口：** 對接學校輔導室。負責撰寫公文、申請「學生比賽/公演公假單」、辦理排練教室借用事宜。
    *   **招生與甄選主持：** 擬定招生簡章；建立 Google 線上報名表；甄選當天主持考生報到、分數統計、公告錄取名單。
    *   **學生保險：** 比賽與公演前，統一向保險公司辦理全團學生的團體意外險。
    *   **指揮對接：** 作為常任指揮之行政助理，統計學生考譜進度、排定課表異動。

## 五、 四大聲部組日常自治與大活動承包機制

家長不以「功能」分組，而是以「孩子所在的樂器聲部」為平日歸屬。

### 1. 日常「聲部自治」規範
*   **聲部代表之職責：** 每個聲部推選 1-2 位聲部代表。負責維護該聲部的 LINE 群組；配合分部指導老師發放與收回樂譜（譜務）；監督聲部學生下課後復原桌椅與譜架。
*   **全員排練值班制：** 學務組與各聲部代表合作，於開學時排定「家長義務值班表」。每位家長每學期需義務值班 1-2 次（每次 2 小時）。值班職責包括：協助點名、準備值班急救箱與飲水、監督下課教室復原、關閉排練室電源與門窗。

### 2. 大活動當天「聲部承包與轉職戰位圖」
大活動當天，家長不聽從臨時指派，而是以「聲部為單位」整組進駐專屬場域，並服從**「活動統籌組」**的現場指揮調度：

```
至德堂 / 比賽會場 (實體空間與聲部戰位對照)
  │
  ├─ [前台大廳] ────> 承包者：【弦樂二部組 (小&中提琴家長)】
  │                    戰位任務：索票登記、驗票入場、貴賓簽到接待、節目單分發。
  │
  ├─ [後台休息室] ──> 承包者：【弦樂一部組 (小提琴一家長)】
  │                    戰位任務：後台學生安全防護、訂購與發放便當、協助換裝化妝。
  │
  ├─ [舞台與卸貨區] ─> 承包者：【大/低音提琴/打擊樂組 家長】
  │                    戰位任務：大貨車裝卸、大型樂器定位、舞台椅位與譜架變更催台。
  │
  └─ [側台與機動區] ─> 承包者：【管樂打擊部部分家長】
                       戰位任務：現場催台帶隊、與音響/錄影技術廠商硬體對接。
```

*   **聲部代表之領導角色：** 大活動當天，各聲部代表即為該功能組的「現場領班」。例如，大提琴聲部代表即為當天的「舞台物流組長」，負責帶領聲部家長執行搬運任務，並隨時與「活動統籌組」保持對講機/LINE 聯絡。

## 六、 招生與甄選標準作業流程 (SOP)

為了確保每年新舊生交接平滑，學務組與活動統籌組應於 **12月 - 2月** 啟動招生甄選，流程規範如下：

```
【步驟 1：招生宣傳 (12月-1月)】
  ├─ 學務組：撰寫招生簡章，送輔導室發文至校內二至四年級各班。
  ├─ 前台公關組 (弦二部)：自主設計電子招募海報，於 FB 及 Line 群組發布。
  └─ 設備組 (大提/打擊)：清點學校可供出借的管樂與弦樂器庫存。
  │
【步驟 2：報名與前置 (1月)】
  ├─ 學務組：製作 Google 線上報名表單，收集報名資料。
  └─ 活動統籌組：協調甄選教室與評審老師時間，編排考場梯次。
  │
【步驟 3：甄選當日 (2月上旬)】
  ├─ 學務組 (弦一部協助)：設立報到處，核對考生身分、發放號碼牌、引導考生進入準備教室。
  ├─ 設備組 (大提/打擊)：佈置甄選教室評審席桌椅、鋼琴及譜架。
  └─ 活動統籌組：現場流程控管，確保每位考生按時進場。
  │
【步驟 4：錄取與報到 (2月下旬)】
  ├─ 學務組：統計評審分數，簽報學校核准後公告錄取名單，發放新生報到通知。
  └─ 財務組：於新生報到日，現場收取新生團費與樂器押金。
```

## 七、 後援會高效營運實戰工具包

為使志工家長不因繁複行政而卻步，後援會必須強制導入並維持以下數位管理工具：

### 1. LINE 官方帳號與群組分流
*   **家長大群組：** 僅用於發布「學務組」或「會長」的官方重要公告（請使用群組記事本）。平日禁止家長在群組內閒聊或傳問候貼圖，避免洗版。
*   **聲部小群組：** 由聲部代表管理，用於平日值班排班溝通、發放分部課譜務通知。
*   **幹部群組：** 僅限會長、財務、活動統籌、學務與各聲部代表，用於即時會務協商。

### 2. Google 雲端協作系統
*   **財務流水帳：** 使用 Google 試算表，由財務主計崗即時登錄。帳冊連結設定為「唯讀分享」，永久放在家長群組公告區，達到「帳目零死角」。
*   **家長值班登記表：** 每學期初由學務組發布 Google 試算表連結，由各聲部家長自行登記平日值班時間，系統自動鎖定額滿時段。

### 3. 工作包 (Job Package) SOP 文件化
*   後援會的各項大型任務均已打包為獨立文件，交接時必須將以下工作包完整移交給下一任幹部，避免重複摸索：
    *   `SOP_01_大件樂器上下大貨車順序與防護手冊.pdf`
    *   `SOP_02_文化中心至德堂前台票務引導工作包.pdf`
    *   `SOP_03_全國學生音樂比賽公文與保險申報模板.docx`
    *   `SOP_04_新團員招生與甄選現場引導流程圖.pdf`
